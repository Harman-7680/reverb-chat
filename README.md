# Reverb Chat Package (Laravel)

A simple real-time **user-to-user chat package** for Laravel using **Laravel Reverb WebSockets**.

It enables instant messaging with private channels and real-time updates.

---

## 🚀 Features

- ⚡ Real-time messaging using Laravel Reverb
- 👤 Private user-to-user chat
- 💬 Simple Blade UI included
- 🔔 Instant message delivery
- 🗄️ Database-based message storage
- 🔒 Secure private channels
- 📦 Auto route registration

---

composer require harman-7680/reverb-chat
